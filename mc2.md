# Mini Challenge 2 - Posts application

- Create a new django app called 'posts' (Add the new app to the INSTALLED_APPS list inside of setting.py).
- Make sure that it has all the correct files. (Create a 'urls.py file inside of posts.)
- In the url.py (inside the config folder), add the path function and configure it.
- Register the new app in settings.
- Create a 'urls.py' file inside of posts and add the includes to the 'urls.py' in the config folder (remember to add a new name for the new endpoints inside of config urls. e.g., "posts/")
- Create the urls.py and register on the parent urls.py (the one from config), and give a name to endpoint 'post/'
Create a new folder called posts inside of templates:
    a. Create a new file for the list.html
    b. Create a new file for the detail.html

    d. Create a new file for the delete.html
